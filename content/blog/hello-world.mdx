### A Taste of Code

Here's a simple health check endpoint in Go — the kind of thing I build every day:

```go
package main

import (
    "fmt"
    "net/http"
    "os"
)

func healthHandler(w http.ResponseWriter, r *http.Request) {
    w.WriteHeader(http.StatusOK)
    fmt.Fprintf(w, "ok")
}

func main() {
    http.HandleFunc("/health", healthHandler)
    fmt.Println("Server starting on :8080")
    if err := http.ListenAndServe(":8080", nil); err != nil {
        fmt.Fprintf(os.Stderr, "Error: %v\n", err)
        os.Exit(1)
    }
}
```

And deploying it is just a few lines of shell:

```bash
# Build and push the image
docker build -t cr.dipak.io/health-check:latest .
docker push cr.dipak.io/health-check:latest

# Deploy to Kubernetes
kubectl apply -f deployment.yaml
kubectl rollout status deployment/health-check
```

You can also reference inline code like `kubectl get pods -n production` or environment variables like `$KUBECONFIG` in prose.
