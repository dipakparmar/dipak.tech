# dipak.tech

Personal portfolio and multi-site platform built with Next.js.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Development

```bash
pnpm install
pnpm dev
```
